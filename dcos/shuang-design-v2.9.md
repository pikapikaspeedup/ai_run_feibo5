# v2.9 爽感大版本 · 割草/多武器/近战 数值重做

> 用户反馈原话："还是不好玩，不够爽。技能还是不够爽，割草还是不够爽，重新做数值设计。
> 武器种类太少，要多种武器各种割草；不要有技能才能多把武器，应该原生支持多把武器；近战武器也要一些。"

## 〇、问题诊断（为什么现在不爽）

1. **多武器被卡在抽卡里**：双持槽要 Lv6 后抽到「双持工牌」卡才解锁——大部分局根本抽不到或抽到时已经决赛圈。
   玩家 80% 的游戏时间只有一把枪，火力形态单一。
2. **副手惩罚过重**：副手只有 55% 伤害，解锁了也像个装饰。
3. **正赛没有草可割**：试用期波次密度是够的（30~80 只/波），但转正后的"琐事骚扰潮"是
   **每 22 秒来 3~6 只、场上上限 14 只**——割草游戏的核心循环在游戏后半段直接消失。
4. **全是远程枪**：12 把主武器全部是射弹/光束，没有一把近战。近战的爽点（贴脸横扫一片、
   击退保龄球、无弹道必中）完全空缺。
5. **技能 CD 太长**：E 槽大招普遍 26~35 秒，一局按不了几次；Q 槽伤害系数偏保守，
   放了感觉不到"清屏"。

## 一、多武器 = VS 模式（四槽体系，v2.9.2 定稿）

> 两轮迭代：初版"开局白送双持"被否（抹掉博弈）；二版"双持卡 + Lv8 编外枪位卡"也被否
> （"双持/Lv8+ 这种奇怪需求哪个游戏这么做"）。定稿 = 吸血鬼幸存者标准模型：
> **武器就是升级卡池里的普通卡——选已有的 = 升一级，选新的 = 多一把，上限 4 把，无任何等级门槛。**

| 槽位 | 伤害系数 | 瞄准 |
|---|---|---|
| 主武器 | 100% | 鼠标/自瞄，可地面换枪/满级融合传说 |
| 第 2 把 | 70% | 与主武器同角度齐射 |
| 第 3 把 | 50% | 独立自瞄最近敌人（随身炮台） |
| 第 4 把 | 50% | 独立自瞄最近敌人 |

- 每次升级抽卡，卡池里最多同时出现两张武器卡：
  **「武器升级」**（随机挑一把已持有未满级的 +1 级，w 1.5）与
  **「新武器上岗」**（还有空槽时随机滚一把未持有武器，每次重掷，w 2.2）——宽 vs 深当场对垒。
- **上桌保底**：连续 2 次一张武器卡都没上桌就强制顶替一张通用技能卡（拿不拿由玩家定）。
- 地面芯片 = 纯运气位：同款升对应槽（四槽都吃）；异款只用于主武器悬停换枪，不开槽。
- 满配 = 100+70+50+50 = 270% 火力，代价是 3 次升级机会不拿技能/模组。

## 三·b 人口控制器（v3.0.1——"怪还是太少"的根治方案）

> 前两轮都在调"刷怪参数"（间隔/每波数量/上限），但 VS 的本质不是"波次刷怪"，
> 是**人口控制器**：每个时间段有一个"场上最低怪数"，杀多快就补多快，屏幕永远是满的。
> 波次只是叠加在人口地板之上的事件尖峰。

### 场上活怪下限（floor）曲线

| 阶段 | floor | 说明 |
|---|---|---|
| 试用期第 m 月 | 16 + 7m（月1=23 → 月5=51） | 波次怪计入人口：倾倒期人口高于地板控制器歇火，波间空窗由控制器填满 |
| 正赛 phase 0/1/2 | 50 / 65 / 80 | 转正即人海 |
| 正赛 phase 3 / 4 | 45 / 30 | 决赛圈收敛，主角让位给同事对决 |
| 硬上限 | **150** | 实测 109 只 update 0.59ms（预算 16.7ms），150 只安全 |

### 控制器规则

- **节拍**：每 0.7s 检查，`alive < floor` 就补 `min(floor-alive, 10)` 只，出生环 380~460px（视野半对角 ≈367px，刚好屏幕外冒出来）。
- **补给类型**：低血冲锋系 fodder（试用期用当月 waveComp 池、正赛用 BR_MOB_POOL，均过滤掉站桩型）；
  后台怪经验打 **7 折**（防等级通胀——密度翻倍击杀翻倍，等级节奏要稳住）。
- **回收器**：每 2s 把跑到 750px 外的普通怪**传送回出生环**（不删不重生，人口守恒；顺带根治"怪卡在远处墙角导致 54/56 卡波"一类问题）。锚点怪不回收（据点/蛇身/编队/站桩）。
- **人墙波保留**：作为 floor 之上的事件尖峰（30+10×phase 只，40~60s 一次）。
- 原正赛"7 秒涓流包"删除，由控制器全面接管。

## 四、技能爽感（actives 全表数值刀）

原则：**Q 槽 CD -15%、伤害 +30%；E 槽 CD -20~25%、伤害 +35%**。大招一局至少按 6~8 次。

## 五、验收口径

1. 转正后 60 秒内屏幕上野怪峰值 ≥ 25 只（旧版 ≤ 14）。
2. 开局 3 分钟内玩家自然拥有 ≥ 2 把武器（旧版需 Lv6 + 抽卡运气）。
3. 机械键盘一刀命中 5+ 只蜂群怪（弧形范围判定）。
4. 人体工学椅挂机站桩对贴身怪持续输出（orbit 无需操作）。
5. E 大招平均 CD ≤ 20s。
