# v3.1 武器扩容设计 ·「打工人玄学科」六件套

> 出发点：VS 的武器多样性不是数值差异，是**几何独占**——每把武器占一种屏幕上的力量形状。
> 我们已占：瞄准线/扇形/抛物线/追踪/链电/回旋/蓄力束/炮台/横扫/环绕/锥喷/短刺。
> 未占的经典几何：贴身光环、随机落点水洼、弹墙台球、随机点名雷击、尾迹地雷、计数爆发。
> 本案六把新武器一人占一个空位，全部画面梗优先；`aura`/`mine` 两个 kind 是引擎现成死代码，直接复活。

## 实现盘点

- `aura`（韭菜盒子）与 `mine`（历史遗留问题）：v18 现成 kind，填 def 字段即用（tickCd/range；cd/triggerR/fuseT/r）。
- `karma`（木鱼）：计数器+周期环，复用 explodeAtNoRecurse + slash 360° 弧光；"功德+1"走 addFloat。
- `stamp`（驳回）：随机抽屏内敌人 → ringwarn 预警 0.4s → 盖章 AoE + 红印 decal（addBattleDecal 现成）。
- `bounce`（踢皮球）：projs 加 wallBounce 分支（世界边界/掩体反弹，vx/vy 镜像，range 按反弹重置）。
- `rain`（空调漏水）：周期在玩家 140px 内随机点落水滴 fx + G.burns 水滩（slow+dps），全复用。
- 图鉴/抽卡/芯片池自动接入（Object.keys(WEAPONS) 驱动）；新阵营标签 'SP'，Hud 国别显示"玄学"。

## 第二组 ·「职场风暴科」六件套（机制派生线：弹射/分裂/火墙/物理锤/剑气/引力）

> 设计原则升级：**机制=梗一体化**——机制本身就是那个职场行为，不是数值贴皮。

### 组合流派（多武器槽的化学反应，白送的深度）

- **物理流**：强制拉会（聚）→ 领导拍板（拍飞互撞）→ 裁员大刀（清走廊）
- **地形流**：火线需求 + 历史遗留问题 + 空调漏水——满地都是不能踩的东西，放风筝天堂
- **无脑站桩流**：电子木鱼 + 韭菜盒子 + 人体工学椅——三圈套一圈，挂机功德割草

### 第二组实现盘点

- ricochet：projs 命中后用现有 p.hit 集合重定向到 160px 内最近未命中敌人（vx/vy 转向），每跳 dmg×1.15。
- split：命中回调里 spawnBullet×3 扇形子弹（dmg×0.5、禁再分裂 flag，Lv5 允许深度 2）。
- firewall：落点垂直方向排 5 个 burns 圆（dps+slow），全复用燃烧区管线。
- slam：swing 变体（窄弧大伤）+ 击飞位移后做一次"落点重叠检测"，撞到的双方各吃冲击伤——shoveWithImpact 小助手。
- slashwave：spawnBullet 新 shape 'wave'（宽判定月牙，pierce 99），出手同时带一次近身 slash 弧光。
- vortex：burns 加 pull 字段（负=吸），燃烧区循环里对圈内敌人施加向心位移；结束时小爆。
- 芯片池稀释提醒：22→34 把后同款芯片概率下降——但升级卡「武器升级」直点已有槽 + 悬停换枪兜底，VS 本身 30+ 把武器同样成立。
